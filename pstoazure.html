Hyper-V to Azure using PowerShell

  • View On GitHub

This Project will take a windows VM from Hyper-V and upload it to Azure. It then adds the VM as an image, and starts a compute instance

Click Here to view the source




This was intended to be a fully automated upload to Azure, but there were some things that could not be automated, Specifically running Sysprep on the VM before it was exported to Azure

The Entire Process can take quite a long time, mainly due to the converting of the VHD file from .vhd to .vhdx, as well as uploading the VM to azure.

Although this script requests inputs several times during the program, These inputs could be preconfigured to allow full automation. The inputs were added to allow flexibility so that others could use the program without modification. Anywhere a needed input could be pulled from Azure instead of requested from powershell it was done to minimize requests

This script could serve as a fantastic template for automation in both the uploading of the VM and also the starting of a VM.

One of the most challenging things to write in this program was a way to traverse the file system with subscriptions, storage accounts, containers, and the objects themselves.
