Trav's Messenger App



This is a Messenger Application using JQuery with the client side hosted on github.

The server was done using SignalR, and hosted on Azure using a Web Application


I had wanted to use WebSockets, however this proved to be an extremely difficult task and SignalR handles all implementations of web communication from WebSockets to Long Polling


The Javascript can be viewed Here and the server files can be viewed here.



This ended up being a long project, and this is the third iteration of a messenger application. The First 2 were done using C# with a windows form application and a console server, using System.Net.Sockets.
